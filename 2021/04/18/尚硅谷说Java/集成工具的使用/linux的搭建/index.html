Centos7的下载

  • 推荐从阿里云的镜像网站上下载。

    • 地址：http://mirrors.aliyun.com/centos/7/isos/x86_64/

  • 点击iso下载即可

Centos7系统的搭建

  • 第一步：创建虚拟机

    • 然后一直点创建

  • 第二步：设置虚拟机

    • 右击，点击设置

    • 将“软驱”移动到最下面，并取消软驱前的选择

    • 选择虚拟盘时：选择自己下载的centos镜像文件

    • 设置网卡

    • 然后点击ok

  • 第三步：启动虚拟机

    • 双击虚拟机即可启动。然后点击确定
    • 选择安装centos

    • 语言选择中文

    • 设置分区：点击后再点击完成即可

    • 然后点击“网络和主机名”。将两个网络连接打开，记住192的开头的IP。

    • 设置完网络后，点击完成，点击开始安装。
    • 设置root密码。输入后如果密码过于简单，只需要连续点击两次完成即可设置成功。

    • 然后等待，最后出现重启按钮后，点击重启即可。

  • 第四步：进入虚拟机

    • 重启后，会出现如下界面。输入账户root。密码为自己上述设置的即可

    • 输入ip addr命令。查看虚拟机的IP

    • 输入dhclient命令，让虚拟机获取得到自己的IP地址。
      • 此时如果不打印任何东西，则获取成功
    • 在输入ip addr命令，就可以看到虚拟机的ip地址

  • 第五步：设置IP为静态IP

    • 进入网络配置文件夹下。输入命令：cd /etc/sysconfig/network-scripts/

    • 查看文件夹下的文件。输入命令：ls

    • 打开网卡的配置文件。输入命令：vi ifcfg-enp0s3
      • 说明：这个ifcfg后面跟的是，在输入ip addr命令后出现192Ip的哪个网卡名。
      • 例如的我网卡名就是enp0s3。
    • 打开配置文件后，按i进入编辑模式，在配置文件中做如下修改
      • 下面的GATEWAY这一行不要写，用自动生成的即可。

    • 然后按ESC,按冒号。在按wq表示退出并保存文件。

    • 重启网络服务。输入命令：systemctl restart network.service

    • 查看IP地址。输入命令：ip addr

    • 测是网络是否连通。输入命令：ping 8.8.8.8

    • 上述都成功时，整个设置就成功了。
